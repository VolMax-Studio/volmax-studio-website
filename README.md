## 🌟 About VolMax Studio Lab

VolMax Studio Lab is a technology engineering company focused on domain-grounded energy ML, power electronics diagnostics, and embedded signal processing. We bridge the gap between heavy industrial electrical work and clean, deterministic software/firmware implementation.

---

## 📦 Core Solutions & R&D

### P1: EnergyMonitor (NILM)
**Non-Intrusive Load Monitoring & Disaggregation**
*   AI-powered residential energy monitor that analyzes the aggregate voltage/current signature at a single point to disaggregate individual appliance states and power metrics.
*   **Status**: Active Development (Portfolio demos live)

### P2: Smart Agriculture
**Autonomous Pump & Irrigation Control**
*   Solar-ready pump control system using the P4 kernel to regulate pressure, eliminate mechanical hammering, and minimize water consumption in remote agricultural applications.
*   **Status**: Prototype

### SmartGrid Monitor
**VPP-ready Grid Anomaly Detection**
*   VPP-ready architecture concept. Energy efficiency monitoring and grid anomaly detection compatible with ENTSO-E CE specifications (simulation study).
*   **Status**: R&D Concept (subject to regulatory market development)

### P4: Neuromorphic Kernel
**Thermodynamic Motor Controller & Symplectic Integrator**
*   Real-time hardware kernel designed for bare-metal microcontrollers (STM32G4 series). Combines symplectic Leapfrog integration and adaptive memory to run deterministic O(1) physics controls at sub-microsecond latency.
*   **Status**: Validation

## 🚀 Getting Started

### Prerequisites
*   Node.js 18+
*   npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/volmax-studio/volmax-studio-website.git

# Navigate to project directory
cd volmax-studio-website

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.
